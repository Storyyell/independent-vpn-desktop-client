# independent_vpn_app

## Project Setup
### Install

```bash
$ npm install
```

### Development

```bash
$ npm run dev
```

### Build

```bash
# For windows
$ npm run build:win

# for macOS and Linux vpn routing rule want to be upated

# For macOS
$ npm run build:mac

# For Linux
$ npm run build:linux
```


## .env file 
```
VITE_SERVER_APP_KEY=<sentinel app key>
VITE_SERVER_API_URL=<sentinel api url>
VITE_MAPBOX_KEY=<mappbox api key>
VITE_IP_API_KEY=<pro-ipapi key>
```
